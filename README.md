# 🎵 가사 영상 만들기 (Lyrics Video Maker)

YouTube 자막을 기반으로 **노래방 스타일 가사 영상**을 자동으로 생성하는 웹 툴입니다.  
YouTube URL만 입력하면 자막 타이밍을 자동 추출하고, 가사를 수정한 뒤 버튼 하나로 **와이프(Wipe) 효과 가사 영상**을 만들 수 있어요.

---

## ✨ 주요 기능

- **YouTube 자막 자동 추출** — URL 입력만으로 타이밍 데이터 자동 수집
- **가사 직접 수정** — 추출된 자막의 텍스트만 정확한 가사로 교체
- **CSV / SRT 파일 업로드** — 직접 작성한 타이밍 파일도 바로 사용 가능
- **와이프(Wipe) 자막 효과** — 흰 글씨 위로 노란색이 왼쪽에서 오른쪽으로 채워지는 노래방 스타일
- **전주 / 간주 표시** — 일정 시간 이상 공백 구간에 자동으로 간주 안내 표시
- **자막 겹침 경고** — 타이밍이 겹치는 구간을 실시간으로 감지하고 하이라이트
- **웹 UI** — 설치 후 브라우저에서 바로 사용

## ⚙️ 설치 방법

### 요구 사항
- Python 3.9 이상
- [FFmpeg](https://ffmpeg.org/download.html) (시스템에 설치되어 있어야 함)

### 설치

```bash
git clone https://github.com/moominIsCute/lyrics-video.git
cd lyrics-video
pip install flask yt-dlp pillow
```

---

## 🚀 실행 방법

```bash
python app.py
```

실행하면 브라우저가 자동으로 열립니다. 안 열리면 `http://127.0.0.1:5000` 으로 직접 접속하세요.

## 📋 사용 방법

### 방법 1 — YouTube URL로 자막 추출

1. YouTube URL 입력 후 **자막 추출** 클릭
2. 추출된 자막 텍스트를 정확한 가사로 수정
3. 영상 파일 업로드
4. **영상 생성** 클릭 → 완료 후 다운로드

### 방법 2 — 파일 직접 업로드

타이밍이 포함된 **CSV** 또는 **SRT** 파일을 직접 업로드해 사용할 수 있어요.

#### CSV 형식 예시
```
번호,시작 시간,종료 시간,자막 텍스트
1,00:00:22;00,00:00:27;15,나의 시선이 나의 마음을 향할 때
2,00:00:27;15,00:00:33;00,자꾸 초라한 모습과 두려움뿐이네
```
